# Let’s Focus  Lose focus much? This release contains support for focus-blur window events.   ## Usage  All you need to do is to add the program option to your application:  go p := tea.NewProgram(model{}, tea.WithReportFocus()) if _, err := p.Run(); err != nil { 	fmt.Fprintln(os.Stderr, "Oof:", err) 	os.Exit(1) }   Then later in your `Update` function, you can listen for focus-blur messages:  go func (m model) Update(msg tea.Msg) (tea.Model, tea.Cmd) { 	switch msg := msg.(type) { 	case tea.FocusMsg: 		// Focused! 	case tea.BlurMsg: 		// Not focused :(         }         return m, nil }   For details, see [WithReportFocus](https://pkg.go.dev/github.com/charmbracelet/bubbletea#WithReportFocus).  ## Tmux  If you're using `tmux`, make sure you enable the `focus-events` option in your config.  config set-option -g focus-events on   Happy focusing (whatever that means)!  ---  <a href="https://charm.sh/"><img alt="The Charm logo" src="https://stuff.charm.sh/charm-badge.jpg" width="400"></a>  Thoughts?
